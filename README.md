## 制作物２📌 **ログイン機能を装備したECサイト** 📌


**概要**  
サーブレット・JSP・SQLを用いたECサイト制作により、Webアプリケーション開発の基礎技術を習得いたしました。
ユーザー認証や商品管理機能の実装を通じて、データベース連携を含む一連の開発プロセスを学びました。

**使用技術**
- 言語: HTML5, CSS3, JavaScript, Java
- フレームワーク: JSP, サーブレット
- データベース: H2 Database (SQL)
- サーバー: Apache Tomcat
- 開発環境: Eclipse
- バージョン管理: Git, GitHub
